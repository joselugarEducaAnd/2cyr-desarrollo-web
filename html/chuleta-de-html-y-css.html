Chuleta de HTML y CSS

Tu chuleta de HTML y CSS

Todas las etiquetas y propiedades que has usado en esta unidad, reunidas en un sitio para consultarlas rápido mientras trabajas.

Etiquetas HTML

EtiquetaPara qué sirve
<!DOCTYPE html>Primera línea. Indica que es una página HTML.
<html> … </html>Envuelve toda la página.
<head> … </head>Información que no se ve (ficha técnica).
<meta charset="utf-8">Para que se vean bien tildes y ñ.
<title> … </title>Título de la pestaña del navegador.
<body> … </body>Todo lo que se ve en la página.
<h1> … <h6>Títulos, de más grande (h1) a más pequeño (h6).
<p> … </p>Párrafo de texto.
<strong> / <em>Negrita / cursiva.
<br>Salto de línea (no se cierra).
<hr>Línea horizontal de separación (no se cierra).
<ul> / <ol> / <li>Lista con puntos / numerada / cada elemento de la lista.
<a href="...">Enlace. Añade target="_blank" rel="noopener" para abrir en pestaña nueva.
<img src="..." alt="...">Imagen (no se cierra). alt = descripción.
<style> … </style>Dentro del <head>: aquí va el CSS.

Propiedades CSS

PropiedadPara qué sirveEjemplo
colorColor del textocolor: red;
background-colorColor de fondobackground-color: #f0f4ff;
font-familyTipo de letrafont-family: Arial, sans-serif;
font-sizeTamaño de letrafont-size: 20px;
text-alignAlineación del textotext-align: center;
margin / paddingEspacio por fuera / por dentromargin: 20px;
widthAncho (p. ej. de una imagen)width: 320px;
transitionCambio suavetransition: all 0.4s;
:hover«Cuando el ratón está encima»h1:hover { color: orange; }
@keyframes + animationAnimación que se repite solaanimation: latido 1.5s infinite;

Recuerda

En CSS, cada propiedad termina en punto y coma ; y el bloque va entre llaves { }. En HTML, casi todas las etiquetas se abren y se cierran (menos <br>, <hr>, <img> y <meta>).

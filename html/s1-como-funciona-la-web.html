S1 · ¿Cómo funciona la web?

¿Qué pasa cuando escribes una dirección?

Cuando escribes una dirección como www.wikipedia.org y pulsas Intro, ocurre algo parecido a pedir un libro en una biblioteca enorme: tu ordenador pide la página y otro ordenador, muy lejos, se la envía.

Los dos protagonistas: navegador y servidor

Para que veas una página web hacen falta dos ordenadores trabajando juntos:

  • El navegador (Firefox, Chrome…) es el programa de tu ordenador. Pide las páginas y las muestra en pantalla.
  • El servidor web es un ordenador especial que está encendido siempre y guarda las páginas. Cuando el navegador se las pide, se las manda.

Esas reglas para pedir y enviar páginas se llaman HTTP (o HTTPS, la versión segura). Por eso las direcciones empiezan por https://.

¿Por qué un servidor y no un ordenador cualquiera?

Porque a una página famosa la visitan miles de personas a la vez. Un servidor está preparado para atender muchísimas peticiones sin apagarse.

Inspeccionar una página real

  1. Abre en el navegador una página que te guste (por ejemplo, la Wikipedia).
  2. Haz clic derecho sobre cualquier parte de la página.
  3. Elige «Inspeccionar» (o pulsa la tecla F12).
  4. Se abrirá un panel con el código HTML de la página. Pasa el ratón por encima de las líneas y verás cómo se iluminan las partes de la página.

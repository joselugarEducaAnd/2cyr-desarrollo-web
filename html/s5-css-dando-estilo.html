S5 · CSS: dando estilo

Ahora, el estilo

Tu página ya tiene contenido, pero se ve «sosa»: letras negras sobre fondo blanco. Con CSS vas a darle color, cambiar los tipos de letra y decidir cómo se coloca todo. Recuerda: HTML pone qué hay; CSS decide cómo se ve.

¿Dónde se escribe el CSS?

La forma más sencilla para empezar es poner una hoja de estilos dentro del <head>, usando la etiqueta <style>. Así todo tu trabajo queda en un solo archivo.

Dónde va el estilo

<head>
  <meta charset="utf-8">
  <title>Mi página</title>
  <style>
    /* aquí van tus estilos */
  </style>
</head>

Cómo se escribe una regla CSS

Una regla CSS tiene tres partes: el selector (a qué etiqueta afecta) y, entre llaves, una o varias propiedades con su valor.

Ejemplo comentado

h1 {
  color: darkblue;
  text-align: center;
}
  • h1 → el selector: esta regla afecta a todos los <h1>.
  • color: darkblue; → una propiedad (color del texto) y su valor.
  • Cada propiedad termina en punto y coma ;.

Las propiedades que más vas a usar

PropiedadPara qué sirveEjemplo de valor
colorColor del textored, #453f93
background-colorColor de fondo#f5f3fc
font-familyTipo de letraArial, sans-serif
font-sizeTamaño de letra20px
text-alignAlineación del textocenter, left
marginEspacio por fuera20px
paddingEspacio por dentro10px

Los colores en CSS

Puedes escribir el color de dos formas: por su nombre en inglés (red, blue, green, orange…) o por su código hexadecimal, que empieza por # y son 6 cifras (#ff0000 es rojo). Con el código puedes usar cualquier color, no solo los que tienen nombre.

Un ejemplo completo para copiar

Página con estilo (cópiala y cambia los valores)

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Mi página con estilo</title>
  <style>
    body { background-color: #f0f4ff; font-family: Arial, sans-serif; }
    h1   { color: #3b3a7a; text-align: center; }
    p    { color: #333333; font-size: 18px; }
  </style>
</head>
<body>
  <h1>Mi página con estilo</h1>
  <p>¡Ahora sí tiene buena pinta!</p>
</body>
</html>

Tarea S5 · Dando estilo a mi página

Añade una etiqueta <style> en el <head> de tu página y, como mínimo:

  1. Un color de fondo para el body.
  2. Un tipo de letra para toda la página (font-family).
  3. Un color distinto para los h1 y para los p.
  4. Alinear algún título al centro (text-align: center;).

Entrega el archivo .html en Google Classroom.

Ampliación (opcional)

Prueba a poner el CSS en un archivo aparte llamado estilos.css y enlázalo desde el <head> con:
<link rel="stylesheet" href="estilos.css">
Así separas del todo el contenido (HTML) de la apariencia (CSS), como hacen las webs profesionales. (Si usas el editor online, quédate mejor con el <style> interno para entregar un único archivo.)

S5 · CSS: dando estilo

Ahora, el estilo

Tu página ya tiene contenido, pero se ve «sosa». Con CSS vas a darle color, cambiar los tipos de letra y decidir cómo se coloca todo. Recuerda: HTML pone qué hay; CSS decide cómo se ve.

¿Dónde se escribe el CSS?

La forma más sencilla para empezar es poner una hoja de estilos dentro del <head>, usando la etiqueta <style>. Así todo tu trabajo queda en un solo archivo.

Dónde va el estilo

<head>
  <meta charset="utf-8">
  <title>Mi página</title>
  <style>
    /* aquí van tus estilos */
  </style>
</head>

Cómo se escribe una regla CSS

Una regla CSS responde a tres preguntas: a qué afecta, qué cambio y cómo.

Las propiedades que más vas a usar

PropiedadPara qué sirveEjemplo
colorColor del textored, #453f93
background-colorColor de fondo#f5f3fc
font-familyTipo de letraArial, sans-serif
font-sizeTamaño de letra20px
text-alignAlineación del textocenter
margin / paddingEspacio por fuera / por dentro20px

Los colores en CSS

Puedes escribir el color por su nombre en inglés (red, blue, green…) o por su código hexadecimal, que empieza por # (#ff0000 es rojo). Con el código puedes usar cualquier color.

Página con estilo (cópiala y cambia los valores)

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Mi página con estilo</title>
  <style>
    body { background-color: #f0f4ff; font-family: Arial, sans-serif; }
    h1   { color: #3b3a7a; text-align: center; }
    p    { color: #333333; font-size: 18px; }
  </style>
</head>
<body>
  <h1>Mi página con estilo</h1>
  <p>¡Ahora sí tiene buena pinta!</p>
</body>
</html>

Tarea S5 · Dando estilo a mi página

Añade una etiqueta <style> en el <head> de tu página y, como mínimo:

  1. Un color de fondo para el body.
  2. Un tipo de letra para toda la página (font-family).
  3. Un color distinto para los h1 y para los p.
  4. Alinear algún título al centro (text-align: center;).

Guarda tu archivo con el nombre s5.html y súbelo a Google Classroom.

Nombre del archivo: s5.html

Ampliación (opcional)

Prueba a poner el CSS en un archivo aparte llamado estilos.css y enlázalo desde el <head> con <link rel="stylesheet" href="estilos.css">. (Si usas el editor online, quédate mejor con el <style> interno para entregar un único archivo.)

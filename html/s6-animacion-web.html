S6 · Animación web

¡Que se mueva!

Una animación hace tu página más viva y llamativa. Hay varias formas de conseguir movimiento en la web; vamos a ver las principales y luego harás la tuya con CSS.

Tipos de animación en la web

TipoQué es
GIF animadoUna imagen que se mueve sola (como los memes). Se inserta con <img>, igual que una foto.
SVG animadoUn dibujo hecho con código que puede moverse. Más avanzado.
Transición CSSUn cambio suave (por ejemplo, al pasar el ratón por encima). Es la más fácil y la que vamos a usar.
Animación CSSUn movimiento que se repite solo, sin tocar nada. Un poco más avanzada.

Mínimo común: una transición al pasar el ratón

Con dos líneas de CSS puedes hacer que algo cambie suavemente cuando el ratón pasa por encima. La palabra clave :hover significa «cuando el ratón está encima».

Un título que crece y cambia de color (cópialo)

<style>
  h1 {
    color: #3b3a7a;
    transition: all 0.4s;   /* el cambio dura 0,4 segundos */
  }
  h1:hover {
    color: orange;
    font-size: 45px;        /* al pasar el ratón, se agranda */
  }
</style>

Pon el ratón encima del <h1> y verás cómo cambia poco a poco. La clave es la propiedad transition: sin ella, el cambio sería brusco.

Extensión: una animación que se mueve sola

Con @keyframes defines un movimiento y con animation se lo aplicas a un elemento. Se repite solo, sin tocar nada.

Un título que «late» (cópialo)

<style>
  @keyframes latido {
    0%   { transform: scale(1);   }
    50%  { transform: scale(1.2); }  /* a la mitad, más grande */
    100% { transform: scale(1);   }
  }
  h1 {
    animation: latido 1.5s infinite;  /* se repite sin parar */
  }
</style>

Consejo de buen gusto

Una animación llama la atención… si no hay veinte a la vez. Con una o dos bien elegidas tu página gana mucho. Si son demasiadas, marea.

Tarea S6 · Mi página se mueve

Incorpora a tu página al menos una animación. Puedes elegir:

  • Mínimo: una transición con :hover (por ejemplo, un título o una imagen que cambie de color o tamaño al pasar el ratón).
  • Extensión: una animación con @keyframes que se mueva sola, o un GIF animado con licencia libre.

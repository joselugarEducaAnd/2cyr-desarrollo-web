S2 · La estructura de una página

Los tres lenguajes de la web

Una página web se construye combinando tres lenguajes, y cada uno tiene un trabajo distinto. Piensa en construir una casa:

  • HTML → la estructura (las paredes, las habitaciones): pone el contenido y lo organiza en títulos, párrafos, listas, imágenes…
  • CSS → el estilo (la pintura, la decoración): colores, tipos de letra, tamaños, dónde va cada cosa.
  • JavaScript → el comportamiento (los mecanismos): hace la página interactiva (botones que reaccionan, juegos…).

En esta unidad trabajaremos sobre todo con HTML y CSS.

¿Qué es una etiqueta HTML?

HTML organiza el contenido con etiquetas. Una etiqueta es una palabra entre los símbolos < y >. Casi todas van por parejas: una de apertura y otra de cierre (la de cierre lleva una barra /).

Anatomía de un elemento

<p>Esto es un párrafo.</p>
  • <p> → etiqueta de apertura (aquí empieza el párrafo).
  • Esto es un párrafo. → el contenido.
  • </p> → etiqueta de cierre (aquí termina).

La estructura mínima de una página

Toda página web parte del mismo esqueleto. Cópialo tal cual: será el punto de partida de todo lo que hagas.

Plantilla mínima (cópiala)

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="utf-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola! Esta es mi página</h1>
    <p>Y este es mi primer párrafo.</p>
  </body>
</html>

¿Qué es cada parte?

PartePara qué sirve
<!DOCTYPE html>Le dice al navegador que es una página HTML moderna. Va siempre la primera.
<html>Envuelve toda la página. El atributo lang="es" indica que está en español.
<head>La «ficha técnica»: información que no se ve en la página.
<meta charset="utf-8">Permite que se vean bien las tildes y la ñ. ¡Muy importante en español!
<title>El título que aparece en la pestaña del navegador.
<body>El «cuerpo»: aquí va todo lo que se ve en la página.

Regla de oro: abrir y cerrar en orden

Las etiquetas deben cerrarse en el orden inverso al que se abren, como cajas dentro de cajas. Fíjate en la sangría (los espacios): ayudan a ver qué va dentro de qué.

Tarea S2 · Mi primera página HTML

  1. Abre el editor (HTML Code Runner o Pluma) y copia la plantilla mínima de arriba.
  2. Cambia el <title> por el título de tu página (por ejemplo, «La web de Marta»).
  3. Cambia el texto del <h1> y del <p> para presentarte y decir de qué tratará tu página.
  4. Comprueba cómo queda (Ejecutar, o guardar y F5) y entrega el archivo .html en Google Classroom.

Ampliación (opcional)

Añade un segundo párrafo y un segundo título con la etiqueta <h2>. ¿Qué diferencia notas entre <h1> y <h2>?
